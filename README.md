ReadTome
========

A tool for converting ebooks into audiobooks with IBM's Watson Cognitive Services TTS API.

<img src="readtome.gif">


Dependencies
------------

- python3
- python3-requests
- python3-ebooklib
- watson-developer-cloud (bundled: https://github.com/watson-developer-cloud/python-sdk)
